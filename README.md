# Evolutionary Autonomous Agents
## Overview
- AI-driven simulation where agents use vision-based food detection and evolve traits through natural selection
## Features
- System for agents to mutate and combine attributes
- Detailed UI
- Gudetama and Princess Batman agents with unique characteristics such as magnetism and projectile launching
## Made Using
- C++
- Object Oriented Programming
- RayLib library
